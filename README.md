# SQL-Based Business Analytics Project

## 📊 Overview

This project analyzes an e-commerce business dataset using SQL to extract meaningful insights such as revenue trends, customer behavior, and product performance.

## 🛠️ Tools Used

* MySQL
* Power BI
* SQL (Joins, Aggregations, Window Functions)

## 📈 Key Insights

* Identified top customers contributing maximum revenue
* Analyzed best-selling products
* Evaluated city-wise sales performance
* Observed monthly revenue trends

## 🚀 Features

* Complex SQL queries (JOIN, GROUP BY, Window Functions)
* Interactive Power BI dashboard
* Real-world business use case

## 📂 Project Structure

* SQL Queries
* Dataset
* Dashboard
* Insights Report

## 💡 Conclusion

This project demonstrates strong SQL and business analytics skills applicable to real-world scenarios.
